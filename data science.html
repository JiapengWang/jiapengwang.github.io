Build model - ARIMA

Description and algorithms

If we combine differencing with autoregression and a moving average model, we obtain a non-seasonal ARIMA model. ARIMA is an acronym for AutoRegressive Integrated Moving Average model (“integration” in this context is the reverse of differencing). The full model can be written as:

$$y′t=c+\phi_1y′t−1+⋯+\phi py′t−p+\theta_1et−1+⋯+\theta qet−q+et$$

where y′t is the differenced series (it may have been differenced more than once). The “predictors” on the right hand side include both lagged values of y′t and lagged errors. We call this an ARIMA(p,d,q) model, where:

p = order of the autoregressive part;

d = degree of first differencing involved;

q = order of the moving average part.

Decide model

    # ARMA(P,0) model ==> AR model
    p = 2
    arma_mod_AR = sm.tsa.ARMA(tr_data,(p,0)).fit()
    AR_aic = arma_mod_AR.aic
    AR_bic = arma_mod_AR.bic
    AR_hqic = arma_mod_AR.hqic
    print('the model akaike information criterion is: %f ' %AR_aic)
    print('the model bayesian information criterion is: %f ' %AR_bic)
    print('the model hannan-quinn criterion is: %f ' %AR_hqic)
    print('---------------------------------------------------')
    print('                                                   ')

    # ARMA(0,q) model ==> MA model
    q = 2
    arma_mod_MA = sm.tsa.ARMA(tr_data,(0,q)).fit()
    MA_aic = arma_mod_MA.aic
    MA_bic = arma_mod_MA.bic
    MA_hqic = arma_mod_MA.hqic
    print('the model akaike information criterion is: %f ' %MA_aic)
    print('the model bayesian information criterion is: %f' %MA_bic)
    print('the model hannan-quinn criterion is: %f ' %MA_hqic)
    print('---------------------------------------------------')
    print('                                                   ')

    # ARMA(p,q) model ==> ARMA model
    p = 5
    q = 2
    arma_mod_ARMA = sm.tsa.ARMA(tr_data,(p,q)).fit()
    ARMA_aic = arma_mod_ARMA.aic
    ARMA_bic = arma_mod_ARMA.bic
    ARMA_hqic = arma_mod_ARMA.hqic
    print('the model akaike information criterion is: %f ' %ARMA_aic)
    print('the model bayesian information criterion is: %f ' %ARMA_bic)
    print('the model hannan-quinn criterion is: %f ' %ARMA_hqic)
